# 2048 Game 🎮

经典的 2048 数字拼图游戏！

## 🕹️ 在线游玩

**点击这里直接玩：** https://glorykim0327.github.io/2048-game-web/

## 🎯 游戏规则

- 使用 **方向键** 或 **滑动** 移动所有方块
- 相同数字的方块碰撞时会合并
- 目标是合成 **2048** 方块
- 游戏在无法移动时结束

## 📁 项目结构

```
2048-game/
├── index.html    # 游戏主页面
├── style.css     # 样式文件
├── game.js       # 游戏逻辑
└── README.md     # 说明文档
```

## 🚀 本地运行

直接在浏览器中打开 `index.html` 即可游玩。
